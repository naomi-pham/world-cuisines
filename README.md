## General info
This project is built with ReactJS, Free Meal API, Tailwind CSS, and daisy UI. 
	
## Technologies
Project is created with:
* [Vite](https://vitejs.dev/guide/)
* ReactJS
* TailwindCSS
* DaisyUI
	
## Setup
To run this project, install it locally using npm:

```
$ cd ../lorem
$ npm create vite@latest 
$ npm run dev
```
